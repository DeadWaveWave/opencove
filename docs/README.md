# Docs README

`docs/` 存放 OpenCove 的专题文档与执行资料。根目录只保留跨专题入口，细分主题放到各自子目录或专题文档中，避免继续把文档地图堆回 `DEVELOPMENT.md`。

## 目录建议

- `agent/`
  - Agent 会话、恢复、切换等专题文档
- `terminal/`
  - 终端架构、迁移、验证与渲染基线
- `cases/`
  - 调试案例、复盘与经验沉淀
